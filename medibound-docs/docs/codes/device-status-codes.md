# Device Status Codes

Device status codes indicate the current state of a device during operation.

| Code  | Display   |
|-------|-----------|
| IDLE  | Idle      |
| STUP  | Needs Setup |
| STRT  | Ready     |
| RXXX / RUNG | Running |
| ERRR  | Error     |
| STOP  | Done      |

Use these codes to track device readiness and detect any errors during device operations.
