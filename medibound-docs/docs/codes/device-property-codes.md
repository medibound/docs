# Device Property Codes

Device property codes help define various characteristics of Medibound devices. Below is the list of all recognized property codes.

| Code  | Display      |
|-------|--------------|
| DESC  | Description  |
| IMAG  | Images of the Device |
| STAG  | Stages       |
| VARS  | Variables    |

These codes are used to assign metadata to devices, such as descriptions or variables that describe behavior and configuration.
