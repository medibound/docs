# Organizational Codes

This document provides details about the different organization types recognized by Medibound and their corresponding codes.

| Code | Display               |
|------|-----------------------|
| OT01 | Manufacturer          |
| OT02 | Software Developer    |
| OT03 | Health and Wellness   |
| OT04 | Fitness and Sports    |
| OT05 | Research and Development |
| OT06 | Security and Privacy  |
| OT07 | Fashion and Design    |
| OT08 | Retail and Distribution |
| OT09 | Education             |

Each code represents an organization type, used to categorize and manage device associations.
