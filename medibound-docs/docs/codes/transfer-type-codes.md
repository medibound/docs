# Transfer Type Codes

These codes indicate how data is transferred between Medibound devices.

| Code  | Display                   | Description                                      |
|-------|---------------------------|--------------------------------------------------|
| LBLE  | Live Bluetooth Low Energy | Real-time Bluetooth Low Energy data transfer.    |
| SBLE  | Static Bluetooth Low Energy | Static data transfer using Bluetooth Low Energy. |
| SNFC  | Static NFC                | Static data transfer using Near Field Communication. |

Transfer types specify how Medibound devices communicate, either using live or static transmission technologies.
