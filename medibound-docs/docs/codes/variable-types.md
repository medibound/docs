# Device Variable Types

Variables types used in devices to represent different data types.

| Code  | Display                                     |
|-------|---------------------------------------------|
| ADEC  | Array Decimal Value (Decimals With Single Spaces In Between) |
| STRG  | Characteristic                              |
| DECM  | Number                                      |

Each code is used to categorize different kinds of variables that a device might need to report or process.
