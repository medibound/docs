# Device Type Codes

The following codes are used to identify different types of Medibound devices.

## Wearables

| Code  | Display          |
|-------|------------------|
| W001  | Health Rings     |
| W002  | Smart Watches    |
| W003  | Fitness Bands    |
| W004  | Smart Glasses    |
| W005  | Activity Trackers |
| W006  | Sleep Monitors   |
| W007  | Heart Rate Monitors |
| W008  | Smart Clothing   |
| W009  | GPS Trackers     |
| W010  | UV Sensors       |
| W999  | Other Wearable   |

## Medical Devices

| Code  | Display                      |
|-------|------------------------------|
| M001  | Blood Pressure Monitors      |
| M002  | Glucose Meters               |
| M003  | ECG/EKG Monitors             |
| M004  | Pulse Oximeters              |
| M005  | Thermometers                 |
| M006  | Respiratory Monitors         |
| M007  | Pain Management Devices      |
| M008  | Mobility Aids                |
| M009  | Diagnostic Imaging Devices   |
| M010  | Drug Delivery Systems        |
| M999  | Other Medical Devices        |

These codes help categorize the different types of devices in the Medibound ecosystem.
