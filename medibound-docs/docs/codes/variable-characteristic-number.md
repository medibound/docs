# Variable Characteristic and Number Types

## Variable Characteristic Types

| Code  | Display                                     |
|-------|---------------------------------------------|
| VC01  | Array Decimal Value (Decimals With Single Spaces In Between) |
| STRG  | Characteristic                              |
| VNXX  | Custom                                      |

## Variable Number Types

| Code  | Display                                     |
|-------|---------------------------------------------|
| VN01  | Array Decimal Value (Decimals With Single Spaces In Between) |
| STRG  | Characteristic                              |
| VNXX  | Custom                                      |

These codes are used to describe variables that represent both characteristics and number data within Medibound devices.
